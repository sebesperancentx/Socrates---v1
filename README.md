# SOCRATES | Oracle d'Alignement Stratégique 🏺

> **"Connais-toi toi-même et tu connaîtras l'univers et les dieux."**

SOCRATES est un sonar psycho-émotionnel de haute fidélité conçu pour transformer le "bruit" de vos journaux intimes et fragments de vie en une **Géométrie du Soi** exploitable pour le pilotage stratégique de votre existence.

## 💎 L'Approche : La Roue Socratique

Inspiré par la méthodologie du **Master IDEA** (Innovation, Design, Entrepreneurship & Arts) développée entre 2009 et 2012, ce projet applique les principes de la maïeutique socratique à l'ère de l'intelligence artificielle.

L'outil ne se contente pas de résumer vos écrits ; il agit comme un processeur qui extrait l'infrastructure invisible de vos comportements à travers trois prismes dialectiques :

1.  **L'Oracle (Philosophie)** : Révèle vos contradictions internes et votre essence par la maïeutique pure.
2.  **L'Architecte (Logique)** : Analyse l'efficience systémique de vos patterns comportementaux.
3.  **L'Alchimiste (Symboles)** : Identifie les archétypes et le potentiel de transformation créative.

## 🚀 Fonctionnalités Clés

-   **Sonar de Fragments** : Importez vos exports JSON de **Day One** ou vos notes brutes.
-   **Géométrie du Soi** : Visualisation dynamique de vos indicateurs de pilotage (Alignement, Charge Cognitive, Densité Métaphorique).
-   **Écho Day One** : Exportation directe des analyses vers votre journal via le protocole `dayone2://`.
-   **Analyse Multi-Prisme** : Basculez entre différentes personnalités d'IA pour varier les angles de vue.
-   **Confidentialité** : Analyse locale et éphémère de vos fragments.

## 🛠️ Stack Technique

-   **Frontend** : React 19, Vite, Tailwind CSS.
-   **Intelligence** : Google Gemini 2.0 (via `@google/genai`).
-   **Visualisation** : Recharts & Framer Motion.
-   **Design** : Esthétique "Swiss Design" minimaliste et typographique.
